Selecting Items

$("selector")

Selecting with jQuery is similar to querySelectorAll.

  • $("tag")
  • $(".class")
  • $("#class")
  • $("parent child")

Manipulating Style

The .css() method is jQuery's interface to styling.

$(selector).css(property, value)

If only a propery is passed, the value of the element is retrieved.
If a value is passed, the property is set to that value.

Methods

$("selector").text(optional arg)

Used to get the text content of (all) selector (and all children). If an argument is passed, the text content is set to the argument. This is analoguous to the native JS .innerText method.

$("selector").html(optional arg)

Analoguous to the native JS .innerHTML method.

$("selector").attr(property, value)

Used to select or set the attribute of a given property to value, if provided.

$("selector").val(optionalvalue)

Get the current value of the first matched element, or set to provided value argument.

$("selector").addClass("classname")

Add class to selected element(s).

$("selector").removeClass("classname")

Remove class from selected element(s).

$("selector").toggleClass("classname")

Toggle class on selected element(s).
