DOM Events

Events:

  • Clicking on a button
  • Hovering over a link
  • Pressing a key
  • Dragging and Dropping

The Process

Select an element, and then add an event listener.

The Syntax

element.addEventListener(type, functionToCall);

var button = document.querySelector("button");

button.addEventListener("click", function() {
console.log("Custom function code here");
});

Using current methods, we could for example override a textContent after a button is clicked.

  • An element can have multiple event listeners
  • An element like a header may be shaped differently than the content. like with a header.
  • An event listener on a UL applies to all LIs
  • Can loop through an HTMLcollection and set event listeners on each. Contextually, can use this.

    for (var i = 0; i < lis.length; i++) { lis[i].addEventListener("click", function() { this.style.color = "pink"; }
