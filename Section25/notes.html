Frameworks vs Libraries

A library gives a set of tools for the user to control to achieve an end.

A framework is like a larger tool with control flow built in, with pre-defined spots for developer entry, like a mad-lib.

The core concept is Inversion of Control: the user has control over a library, while the framework has "controL over the user.

Heavy vs Light Frameworks

Heavy frameworks have more "control" versus light frameworks.

Rails is a heavier framework, express is a lighter framework.
