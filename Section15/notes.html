Methods

Five Primary Methods for selecting Elements

  • document.getElementById() - Takes a string argument and returns the first (ideally, only) element with matching ID.

  • document.getElementsByClassName() - Takes a string argument and returns an HTMLCollection of elements that have a matching class.

  • document.getElementsByTagName() - Takes a string argument and returns an HTMLCollection of elements that have a matching tag type.

  • document.querySelector() - Takes a string argument of any CSS-style selector and returns the first matching element.

  • document.querySelectorAll() - Takes a string argument of any CSS-style selector and returns an HTMLCollection of all matching element.

Style

The style property accesses the CSS styling of an element.

var tag = document.getElementById("example")

  • tag.style.color
  • tag.style.border
  • tag.style.fontSize

classList

While elements can be individually styled one style element at a time, it's DRYer to to change an element's class.

var tag = document.getElementById("example")

  • tag.classList.add("className")
  • tag.classList.remove("className")
  • tag.classList.toggle("className")

textContent

Returns a string of all the text contained in a given element. This allows us to retrieve text content, but also set the text content. Note: this will also wipe out any tags nested inside the tag. If attempting to overwrite content with tags, then use innerHTML.

var tag = document.querySelector("p");

tag.textContent - retrieves text content

tag.textContent = "new text content" - sets new text content

innerHTML

Returns a string of all the HTML contained in a given element. This is similar to textContent, but allows us to override content with HTML, not just plaintext.

Attributes

Use getAttribute() and setAttribute() to read and write attributes like src or href.

var link = document.querySelector("id");

link.getAttribute("href")

link.setAttribute("href", "www.google.com");
