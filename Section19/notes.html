Events

$('selector').click(function() {})

Add click listener to all selected element(s).

$('selector').

$(this)

Keyword for selecting current element. e.g. $("button").click(function() { $(this).css("background", "pink")})

$('selector').keypress()

Triggered right after key press, and returns the actual key pressed.

$('selector').keydown()

Triggered right after key press, and returns a code based on the actual key pressed.

$('selector').on(event, function)

Analoguous to the .addEventListener() method.

  • click() only adds listeners for existing elements.
  • on() will add listeners for all potential future elements.
