Events

$('selector').click(function() {})

Add click listener to all selected element(s).

$('selector').

$(this)

Keyword for selecting current element. e.g. $("button").click(function() { $(this).css("background", "pink")})

$('selector').keypress()

Triggered right after key press, and returns the actual key pressed.

$('selector').keydown()

Triggered right after key press, and returns a code based on the actual key pressed.

$('selector').on(event, child, function)

Analoguous to the .addEventListener() method.

  • click() only adds listeners for existing elements. For example, if we add new elements to a page they don't gain the click events.
  • on() will add listeners for all potential future elements.
  • This is done by setting the event on the parent, and then specifying as argument the children elements (including future children elements) which the event should apply to.

Effects

$('selector').fadeOut()

Fades out selected element(s).
